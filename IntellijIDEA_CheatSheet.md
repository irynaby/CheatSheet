#Intellij IDEA - Cheat Sheet (Краткая шпаргалка)

### Поиск и замена
``` bash
Ctrl + F	# Поиск
F3	# Поиск вперед
Shift + F3	# Поиск назад
Ctrl + R	Замена
Ctrl + Shift + F	Поиск в проекте
Ctrl + Shift + R	Замена в проекте
Ctrl + Shift + S	Поиск используя шаблон
Ctrl + Shift + M	Замена используя шаблон
```
